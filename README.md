# d2city
```
1. Generate bdd annotation in coco format : python bdd2coco.py -l label_path -s label_path
2. Generate bdd annotation in coco format (6 shared classes with d2city) : python bdd2coco_label_convert.py --l label_path -s label_path
3. Generate d2city annotation in coco format (6 shared classes with bdd) : python dd2coco_det_label_convert.py -dir label_path
```
